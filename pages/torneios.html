TORNEIOS

Introdução

Nesta página estarão disponíveis todas as informações sobre os Torneios


Para ajudar na sua busca, aconselhamos que você use o comando Ctrl + F

Observação: As premiações são entregues automaticamente aos vencedores dos torneios, consultem sempre seu inventário após a finalização.

Torneios

Os Torneios são feitos especialmente para os jogadores que curtem batalhar em grandes arenas de PvP.

Ao realizarem o diálogo com o NPC ele lhes cobrará o valor de 20HDS para que entre na arena

Após se inscrever no NPC, os participantes são enviados para uma sala de espera, até que o torneio realmente comece.

Depois do tempo de espera, os participantes são levados para uma arena de batalha, onde eles devem lutar entre si até restar apenas um jogador, o campeão do torneio.

Podendo ser encontrado diretamente no PVP, o NPC Nick levará todos os treinadores à arena

Premiações

Os vencedores do torneio receberão todos os hds investidos para inicia-lo + uma bonificação por vencer
